BoiPoka is a modern Full Stack Progressive Web Application (PWA) for reading and managing eBooks. Users can discover books, search by title, author, or category, organize books into Read and Wishlist collections, and enjoy an app-like reading experience. Administrators can manage books, upload PDFs to Google Drive, and control user roles through an intuitive dashboard.

# 🚀 Installation

Clone the repository

```bash
git clone YOUR_CLIENT_REPOSITORY
```

Go to project

```bash
cd boipoka-ebook
```

Install dependencies

```bash
npm install
```

Run

```bash
npm run dev
```
